## 2. 动画库许可证要点（避坑）

- **GSAP**：核心 + 全部原 Club 插件（**ScrollTrigger / SplitText / Flip / DrawSVG / MorphSVG / ScrollSmoother / Inertia**）**全部免费含商用**。⚠️ 非开源：走 GreenSock "standard no-charge license"，禁反编译、禁把 GSAP 嵌入"无代码动画构建器"这类与 Webflow 竞争的产品、禁重分发为自有库。**对教学网站无影响，可放心全量用。** 用 `@gsap/react` 的 `useGSAP()` hook 管理生命周期（自动 cleanup + `contextSafe`，避免 StrictMode 双挂载泄漏）。
- **Motion**：`motion/react` 导入（不再是 `framer-motion`），**MIT 完全免费**，React 19 兼容。付费的只是 AI Kit / premium components，核心库不需要。
- **anime.js v4**：MIT，全面 ESM，tree-shaking 友好；**优先 GSAP/Motion，anime.js 仅按需**。
- **react-bits**：**MIT + Commons Clause**——可免费用于教育/商用、可改可分发；唯一禁止=把"这个库本身"作为产品出售。**本站合规**。非 npm 依赖：`npx shadcn add @react-bits/<Comp>` 或直接复制源码进仓库（代码归你维护）。部分背景件依赖 `ogl`/`three`，注意体积。
- **r3f + drei**：MIT。React 19 → 必须 `@react-three/fiber@9+` + `@react-three/drei@10+`（版本对齐否则报错）。
- **KaTeX / react-katex / Shiki / Lucide**：均 MIT。字体 OFL 1.1。音效 CC0（Freesound 需**逐条**确认 CC0，CC-BY 要署名）。

## 3. 关键交互 · 推荐实现

### 3.1 DP 表格逐格填充 + 高亮 + 改值重播（全站核心引擎）
- **载体**：**DOM/CSS Grid 真实单元格**（非 Canvas）→ 可无障碍、tooltip、公式叠加。
- **计算与动画解耦**：把 DP 求解写成**步骤生成器** `steps[]`，每步 `{cell:[i,j], value, deps:[...], formula}`；UI 只按 step 索引渲染。
- **动画**：GSAP timeline 按 steps 顺序补间（`.to` + `.stagger`）；播放/暂停/单步/拖进度条绑 `timeline.progress()`。当前格=高饱和描边+微 scale，依赖格=次级色+连线。
- **改输入重播**：输入变→重跑生成器得新 steps→`timeline.kill()` 重建；用 `useGSAP()`（依赖含输入，自动清理）。
- 想要"填格时整行重排"用 GSAP **Flip** 或 Motion `layout`。

### 3.2 状态转移依赖箭头 / 转移图
- **表格内箭头**：SVG overlay 一层绝对定位盖在 Grid 上，`<path>` 画依赖→当前的曲线箭头；出现动画用 DrawSVG（已免费）或 `stroke-dashoffset`。
- **独立 DAG / 转移图**：小规模手绘 SVG；节点多用 d3-force 布局或 **React Flow**（可拖拽）。
- **树形 DP**：2D 用 SVG 树；炫酷 3D 展开才上 r3f。

### 3.3 几何动态渐变外壳（三层，WebGL 仅 hero 一处）
| 层 | 方案 | 笔记本表现 | 手法 |
|---|---|---|---|
| 氛围层（多数区块） | **CSS 渐变 + 动画**（conic/linear + `@property` 补间 + blur + SVG 噪声） | ✅ 最省，GPU 合成 | 动画 `--angle`；`feTurbulence` 防 banding |
| 结构层（主视觉几何） | **SVG 几何**（多边形/网格/线条构成） | ✅ 矢量清晰可动 | GSAP 补间点位/描边 |
| 高级流动（**仅 hero**） | **WebGL shader mesh**（react-bits `Aurora`/`Threads` 或轻量 fragment noise） | ⚠️ 吃 GPU，需节流 | `dpr≤1.5`、离屏暂停 |

**铁律**：渐变几何只做外壳/背景，正文区切近纯色高对比；所有动态背景**离屏即暂停**（IntersectionObserver）；`prefers-reduced-motion` 降级为静态渐变图。**同屏最多一个 WebGL 上下文**。

### 3.4 滚动分步揭示（scrollytelling）
- 首选 **GSAP ScrollTrigger**（`scrub`+`pin`）：把 DP 推演钉住，随滚动逐步点亮公式行/填表格。
- 备选 Motion `useScroll`+`useTransform`（原生 ScrollTimeline，更省主线程），适合纯淡入/视差。
- 降级：`prefers-reduced-motion` 关 pin/scrub，退化为点击"下一步"。

### 3.5 互动小游戏载体
- **默认 SVG+DOM**（与讲解表格同构、可无障碍、易拖拽高亮）。
- 单屏动态元素数百且需连续 60fps 重绘 → **Canvas2D**。
- **WebGL 只留给 3D 演示**，一般小游戏不需要。

## 4. 免费素材清单

### 4.1 字体（全 OFL 1.1，免费含商用）
| 字体 | 角色 | 说明 |
|---|---|---|
| **Noto Sans SC / 思源黑体 SC** | 正文黑体（首选） | Pan-CJK；**必须 woff2 子集化** + `font-display:swap` |
| Noto Serif SC / 思源宋体 | 长文/引文（可选） | 衬线阅读变体 |
| **得意黑 Smiley Sans** | 标题/展示体 | 窄+斜 6°+几何人文平衡，覆盖 GB2312 全 6763 字，官方 .woff2 |
| Space Grotesk / Sora / Onest | 拉丁标题/数字 | 配得意黑做中英混排 |

⚠️ **CJK 字体巨大**（思源全量数 MB）：**必须 `pyftsubset` 子集化**（按站内实际用字含术语表生成 woff2）或 `unicode-range` 分片；正文加系统字体栈兜底。**这是包体积最大的杠杆。**

### 4.2 图标（全 MIT）
- **Lucide** 为主（1500+，`lucide-react` 具名导入 tree-shake 最佳）。
- 缺图标从 **Tabler**（5000+）/ **Phosphor**（9000+/6 字重）补单个，均 MIT 可混用。

### 4.3 渐变 / 噪声 / 纹理
- 渐变生成器（导出代码，非依赖）：gradients.design、Stripe 风 mesh-gradient 开源实现（MIT 自托管）。
- 噪声/grain：**SVG `feTurbulence`**（原生零资源）或程序化 GLSL noise。
- CC0 位图纹理（少用以控体积）：Poly Haven、ambientCG（均 CC0）。
- 配色工具：coolors.co / realtimecolors.com（产出配色不涉版权，落成 CSS 变量）。

### 4.4 音效（CC0 优先）
- **Kenney UI Audio / Interface Sounds**（**CC0 无需署名**）——首选，专为 UI 反馈。
- Freesound 筛 CC0 标签（**逐条确认**，CC-BY 需署名）；Mixkit/Pixabay/Sonniss 补充（读条款）。
- 工程：音频懒加载 + 用户手势后解锁（自动播放策略）+ 静音开关。

### 4.5 可选可视化素材
- React Flow（MIT）转移图/DAG；D3（BSD）布局与坐标；unDraw（免费商用，几何扁平插画）做空状态/章节插图。

**已排除**：任何需付费或禁再分发的字体/音效/图库。

## 5. 工程结构与性能

### 5.1 Vite 项目结构（建议）
```
src/
  routes/            # 每 part/章节一路由（React Router lazy 懒加载）
  features/dp-engine/# DP 步骤生成器 + 表格/数组/树视图 + 动画（全站复用核心）
  features/viz/      # SVG 依赖箭头 / 转移图 / 3D 树（独立 chunk）
  features/hero/     # 几何渐变外壳（WebGL 单点，独立 chunk）
  features/games/    # 7 个互动小游戏
  content/           # MDX/Markdown 题解（构建期 Shiki 高亮 + KaTeX 预渲染）
  shared/ui/         # 布局原语、代码面板、题目卡、公式、信息框
  tokens/            # 设计令牌（CSS 变量：配色/字体/间距/7 部分渐变身份）
```

### 5.2 性能红线（笔记本 1366–1536 / 60fps）
- **代码分割**：路由级 `React.lazy`+`Suspense`；重依赖单独 chunk 且动态 `import()`——`three`/`@react-three/*`（3D 树）、`ogl`（react-bits 背景）、GSAP 插件按需。Vite `manualChunks` 把 three/gsap 分包，不进主 bundle。
- **KaTeX 构建期预渲染**：`rehype-katex` + `katex.renderToString` 打成静态 HTML；页面只引 `katex.min.css`（子集化字体），运行时 0 KaTeX JS。
- **Shiki 构建期预高亮**：`@shikijs/rehype`/`rehype-pretty-code`，`langs` 只列 `cpp`，`themes` 固定 1–2 个；用户侧 0 高亮 JS。
- **动画**：只碰 `transform/opacity/filter`；`will-change` 用完即撤。GSAP 用 `gsap.matchMedia()` 分支，Motion 用 `useReducedMotion()`。
- **WebGL/r3f**：`dpr` 上限 1.5、`frameloop="demand"` 或离屏 IntersectionObserver 停帧、单场景、LOD、**用 `useFrame` ref-mutate 绝不每帧 setState**、`useMemo` 缓存 geometry/material；同屏最多一个 WebGL 上下文。
- **包体积**：图标具名导入；GSAP 只注册用到的插件；**CJK 字体子集化是最大杠杆**（数 MB→数百 KB）；`rollup-plugin-visualizer` 定期盯；静态产物开 gzip/brotli。
- **产出**：`vite build → dist/`，直接扔任意静态服务器/CDN 即可部署。
