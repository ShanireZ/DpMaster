## M1.5 设计反馈迭代（owner 评：配色太土/机器化、讲解太简略、latex 有问题）
- **配色重做（Warm Ink）**：弃"深靛底+紫→青→琥珀彩虹渐变"（典型 AI 默认）→ 暖中性近黑底 `#0b0a09` + 蜜金主色 + 七部分**精选低饱和宝石色**（青/琥珀/玫/翠/靛/橄/紫，同一套和谐非彩虹）；发丝描边(rgba)、柔和弥散阴影、内高光双层卡、细描边图标、on-accent 暖黑文字(可读)；浅色模式改**暖奶油** `#f4f1ea`（非冷灰）。几何背景改单色调（去硬编码青、去彩虹）。用 high-end-visual-design 规范纠偏。
- **KaTeX 真 bug 修复**：`react-katex@3.1` 与 `katex@0.17` 不兼容 → `\max\big` 当字面文本渲染。改 `Math.tsx` **直连 `katex.renderToString`**（去 react-katex）；公式内中文移出 `\text{}`（KaTeX 数学字体无 CJK）。89 处公式全正常、零反斜杠残留。
- **内容深化（图文并茂+逐步）**：01 背包加**场景插图**(物品+背包 SVG)、**贪心反例**(贪心 7 vs 最优 9)、**取舍决策分叉图**、**"跟着算一遍"4 步卡**、指向演示/游戏的引导条；section 从 5 增至 7。
- **动效**：★坑=无头/后台标签页会**暂停 CSS 动画与 IntersectionObserver**，凡从 `opacity:0` 起步的进场都会把内容卡成隐藏态（两次踩到）→ **移除淡入进场**，只留 hover/press 微交互，内容始终可见优先。
