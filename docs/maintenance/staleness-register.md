# Recorded Risks

* `site/wrangler.jsonc` is required for Cloudflare Workers Static Assets. It should be tracked or recreated exactly from the root deploy runbook on fresh clones.
* EdgeOne's catch-all edge function path is generated by `site/scripts/postbuild.mjs`; its behavior should be tested after deploy because older notes called part of the route behavior unofficial.
* Feedback success means the structured `feedback_received` log was written. DingTalk forwarding remains best effort and is diagnosed separately through `feedback_webhook` events.
* The built-in 30-minute/10-request feedback limiter is isolated per edge instance. Global enforcement requires matching platform WAF or rate-limit rules.
* The problem corpus relies on public Luogu metadata. Re-check problem availability, difficulty, and native status during major content refreshes.
